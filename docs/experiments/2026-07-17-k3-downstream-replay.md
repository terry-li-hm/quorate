# K3 downstream synthesis replay

## Decision

Retain K3 in the Kimi seat, but do not claim that its blind-phase advantage has been proved to improve final Quorate syntheses. The controlled downstream result was valid and inconclusive.

## Method

Six fixed, synthetic, non-sensitive decisions represented security, architecture, planning, evidence, operations, and governance. Five peer models produced one set of blind claims per case. Those claims were held identical while only the Kimi seat changed between K2.6 and K3. Each candidate supplied a blind claim and one debate intervention. Fable 5 and GPT-5.6 Sol then synthesized each condition independently, and Opus 4.8 compared the anonymous synthesis pairs against a hidden critical finding.

The locked primary metric awarded K3 one point for a downstream synthesis win, half a point for a tie, and zero for a loss. Eight of twelve possible points were required to establish downstream advantage. Four or fewer points across all twelve completed comparisons would have reverted the seat. Route availability, fixed-peer quorum, paired synthesis completion, and evaluator contracts were non-compensable floors.

## Result

Five cases yielded ten completed comparisons. K2.6 returned one empty OpenRouter response on the capacity-forecast case, removing both synthesis comparisons for that case. Every predeclared validity floor still passed.

K3 earned 6.0 points across the ten completed comparisons. On Fable syntheses, K3 won four comparisons and K2.6 won one. On GPT syntheses, K3 won two and K2.6 won three. There were no ties. This directional split is exactly the kind of result the material threshold was designed not to overinterpret.

The result did not clear the eight-point downstream-evidence threshold, but it also could not trigger reversion because the replay was incomplete and did not materially favor K2.6. K3 therefore remains in Quorate based on its decisive blind-phase result and stronger route reliability, while downstream benefit remains unproved.

The aggregate report is `~/.local/state/quorate/council-seat-assays/2026-07-17-k3-downstream-replay.json`. It retains no model responses. The one-off harness was not promoted to maintained benchmark code because the experiment did not materially improve the roster decision.

## Limitation

Holding peer claims fixed isolates the Kimi seat, but it cannot measure reciprocal effects in which K3 changes later peer arguments. Allowing peers to regenerate would capture that interaction while confounding the single changed variable, so this replay deliberately chose causal isolation over ecological completeness.
